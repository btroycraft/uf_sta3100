This repository contains the quarto documents necessary to build the class materials for UF STA3100.

Dependencies:
1) [R](https://cloud.r-project.org/)
2) [Quarto](https://quarto.org/docs/get-started/)
3) TinyTeX - available via `quarto install tinytex` after first installing Quarto.

Project builds by running `quarto render` in the root directory.
